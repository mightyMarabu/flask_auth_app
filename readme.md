# create app

## npm package manager

execute in 'project'-folder
```
npm init -y
```
## start app
### server
```
source auth/bin activate
export FLASK_APP=project
export FLASK_DEBUG=1

```
### client
execute in 'project'-folder
```
npm run build
```
